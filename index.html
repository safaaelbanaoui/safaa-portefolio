Projet R Analyse de cancer

Analyse des données avec R pour examiner la corrélation entre l'incidence de certains cancers et le continent d'Afrique ou d'Europe en mettant en lumière les disparités continentales dans la charge de morbidité et explorant les facteurs sous- jacents.

Développement d'un code MatLab pour visualiser la distribution d'énergie et les cartes de température, utilisant le modèle de Monte Carlo pour estimer les profils de température et les zones affectées par la chaleur dans le contexte de l'ablation laser pour le cancer du sein.
